help & rules

  1. the goal is to guess dog breeds based on the image and acquire as many points as possible
  2. once started, there is an option to start a timer. to keep things fair, this option goes away after you answer the first question
  3. if the timer is started and you save your score, your time data will also upload to the leaderboard
  4. each question is worth 50 points. no deductions for incorrect answers
  5. have fun!
