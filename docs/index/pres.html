CPSC 332 Final Project Presentation

Dog Gusser

Matt Nguyen

For my CPSC332 final I chose to build a web app called Dog Guesser. This app is similar to Geoguesser/Kahoot where users are presented with an image of a dog gotten from the Dog CEO API and four mutiple choice answers. The aim of this app is to provide users with a fun, slightly-competitive game where users can learn more about dogs breeds while playing

Functional Requirements:

  1. Users must be viewing a page appropriately styled using CSS toward the theme: dogs (front)

  2. Users must be able to click a start-game button (front)

  3. Users must be able to select an option for a timed game (front)

  4. Users must view questions gotten from the Dog CEO API (front + back)

  5. Users must be able to view an in-game progress bar (front)

  6. Users must be able to see their score update live (front)

  7. Users must be able to input a chosen username that will be saved (front + back)

  8. Users must be able to view their time (if applicable) at the end of the game (front + back)

  9. Users must be able to view the scores of other users (front + back)

  10. Users must be able to view times (if applicable) of other users (front + back)
